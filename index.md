## 🚀 Optimized settings

Performance matters when working with high-polygon models. You can tune the software to match your hardware capabilities. Go to Options and select View. Change the display mode to Rendered for a preview of lights and materials. If the view feels slow, switch back to Shaded mode to save system resources.

Consistent backups prevent frustration. Set your auto-save interval to 10 minutes within the Options menu. This ensures that the software periodically writes a recovery file in case your computer loses power or the application closes unexpectedly.

Use the Properties panel to view detailed data about any object you select. This includes information about the geometry type, layer name, and material properties. This metadata helps in keeping your files organized throughout the design process.

When working with large structures, group your components early. Select multiple parts and press Ctrl+G to create a group. This simplifies the act of moving complex assemblies across your workspace without affecting individual points. Use the Explode button if you need to return parts to their basic shapes later.
